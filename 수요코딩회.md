# 🧠 수요 코딩회 (수요일)

## 🎯 목적

- 단 하루, AI를 도구 삼아 팀별 프로젝트에 몰입합니다.
- 결과물뿐만 아니라 과정도 중요합니다.  
  AI가 작성한 코드를 완벽히 내 것으로 소화해 그 원리를 설명할 수 있어야 합니다.
- 낯선 기능도 Top-down으로 빠르게 풀어내며 **학습 민첩성**을 극대화하고,  
  현업 레벨의 기술을 직접 부딪쳐보며 실무 감각을 익힙니다.

---

## 📌 요구사항

- 이번 주 과제: **미니 DBMS - API 서버 구현**
- 구현한 API를 통해 **외부 클라이언트에서 DBMS 기능 사용 가능**해야 함
- **스레드 풀(Thread Pool)** 구성
  - 요청마다 스레드를 할당하여 SQL 요청을 **병렬 처리**
- 이전 차수에서 구현한:
  - SQL 처리기
  - B+ 트리 인덱스  
    👉 그대로 활용하여 내부 DB 엔진 구성
- 구현 언어: **C 언어**

## ⚡ 개발 방향

- “학습”보다 **“구현” 우선**
- AI 적극 활용 (코드 생성 포함)
- 단, 아래는 반드시 충족해야 함:

### ✔️ 필수 조건

- 생성된 코드 **이해 및 설명 가능해야 함**
- 핵심 로직은 반드시 숙지

### ✔️ 허용 범위

- 코드 전체를 AI로 생성해도 OK
- 하지만 **검증 + 이해는 필수**
